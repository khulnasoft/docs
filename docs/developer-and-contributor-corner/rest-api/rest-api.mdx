# API

## Khulnasoft Agent REST API

The complete documentation of the Khulnasoft Agent's REST API is documented in the OpenAPI format [in our GitHub repository](https://raw.githubusercontent.com/khulnasoft/khulnasoft/master/src/web/api/khulnasoft-swagger.yaml). 

You can explore it using the **[Swagger UI](https://docs.khulnasoft.com/api)**, or the **[Swagger Editor](https://editor.swagger.io/?url=https://raw.githubusercontent.com/khulnasoft/khulnasoft/master/src/web/api/khulnasoft-swagger.yaml)**.

## Khulnasoft Cloud API

A very basic Khulnasoft Cloud REST API supports the [Grafana data source plugin](/docs/dashboards-and-charts/grafana-data-source-plugin), 
but has not yet been expanded for wider use. We intend to provide a properly documented API in the future.
