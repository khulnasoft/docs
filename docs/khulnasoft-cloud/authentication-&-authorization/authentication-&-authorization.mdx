# Authentication & Authorization

This documentation covers the authentication methods available in Khulnasoft Cloud and explains how authorization controls access and permissions for team members.

## Authentication

| Method         | Description                                                                                                                                                                                                                   | Setup Process                                                                                                              |
|:---------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| Email          | • Standard email and password authentications<br/>• Recommended for individual user                                                                                                                                           | 1. Visit Khulnasoft Cloud<br/>2. Enter email address<br/>3. Follow verification process<br/>4. Set up password (new accounts) |
| Google OAuth   | • Authentication using Google account credentials<br/>• Account will be linked to your Google email address                                                                                                                   | 1. Visit Khulnasoft Cloud<br/>2. Click Google sign-in<br/>3. Complete Google authentication flow                              |
| GitHub OAuth   | • Authentication using GitHub account credentials<br/>• Account will be linked to your GitHub email address                                                                                                                   | 1. Visit Khulnasoft Cloud<br/>2. Click GitHub sign-in<br/>3. Complete GitHub authentication flow                              |
| Enterprise SSO | • Advanced authentication for organizations using identity providers<br/>• Features:<br/>&emsp; - Identity provider integration<br/>&emsp; - Centralized management<br/>&emsp; - Enhanced security<br/>&emsp; - Audit logging | See [Enterprise SSO documentation](/docs/khulnasoft-cloud/authentication-&-authorization/enterprise-sso-authentication)  |

> **Important**: When using OAuth, your Khulnasoft Cloud account will be automatically associated with the email address provided by the OAuth provider. Ensure you have access to this email address.

## Authorization

Once logged in, you can manage role-based access in your Space to give each team member the appropriate role. For more information, see [Role-Based Access model](/docs/khulnasoft-cloud/authentication-&-authorization/role-based-access-model).
