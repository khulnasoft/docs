# Khulnasoft Cloud Versions

Khulnasoft Cloud is available in two versions:

| Version                | Description                                                                                                                                                                                                                     |
|:-----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Cloud Service**      | The fully-managed service hosted at [app.khulnasoft.com](https://app.khulnasoft.com). [Connect](/docs/khulnasoft-cloud/connect-agent-to-cloud) your Agents to access comprehensive infrastructure monitoring, real-time dashboards, and advanced features. |
| **Enterprise On-Prem** | A self-hosted version of Khulnasoft Cloud for businesses requiring complete control over their monitoring infrastructure. Includes all features of our Cloud offering, deployed within your own environment.                       |

Visit our [Pricing](https://www.khulnasoft.com/pricing/) page to compare features and options.

## Using Khulnasoft Cloud Service

1. [Create an account](https://app.khulnasoft.com) on Khulnasoft Cloud.
2. Follow the provided setup instructions to install and connect Agents to your Space automatically

## Installing On-Prem

To docs more about installing Khulnasoft Cloud in your own infrastructure:

1. Review the [Architecture Diagram](/docs/khulnasoft-cloud/versions/on-prem) to understand the system components.
2. Follow the [Deployment Guide](/docs/khulnasoft-cloud/versions/on-prem/installation) for detailed installation instructions
